## Setup

Make sure to install the dependencies:

```任意のシェル
cd yi

# npm
npm install

# pnpm
pnpm install

# yarn
yarn install
```

## Development Server

Start the development server on `http://localhost:3000`:

```任意のシェル
cd yi

# npm
npm run dev

# pnpm
pnpm run dev

# yarn
yarn dev
```
